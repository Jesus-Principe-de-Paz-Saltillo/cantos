# Repositorio de Cantos – Misión Jesús, Príncipe de Paz

Este repositorio funciona como la **fuente de verdad** de los cantos utilizados por el ministerio de música de la **Misión Jesús, Príncipe de Paz**. Aquí se concentran los archivos de los cantos en formato **ChordPro**, de manera que cualquier integrante del ministerio (o colaborador externo) pueda consultarlos, editarlos y visualizarlos con el software que prefiera.

La intención no es imponer una sola forma de uso, sino **centralizar el contenido** y permitir que cada quien lo consuma desde su herramienta favorita.

---

## 1) ¿Para qué es este repositorio?

### 1.1) Resumen rápido

- **Propósito:** Repositorio central de cantos del ministerio de música
- **Formato:** ChordPro
- **Idioma principal:** Español (no limitado a otros idiomas)
- **Versionado:** No se manejan versiones formales; el repositorio es un documento vivo
- **Uso:** Interno del ministerio, pero de acceso público

Aquí se almacenan los cantos tal como el ministerio los utiliza: letras, acordes y estructura musical, manteniendo consistencia y evitando múltiples copias divergentes.

### 1.2) Aviso legal y uso de contenido (Disclosure)

Este repositorio tiene **fines exclusivamente ministeriales, educativos y de servicio**, sin ningún objetivo comercial.

Los cantos aquí contenidos pueden incluir composiciones originales, cantos de dominio público o adaptaciones de obras cuyos derechos pertenecen a sus respectivos autores y/o editoriales. En los casos donde aplique:

- **No se reclama autoría** sobre letras o melodías que no sean propias.
- **No se pretende sustituir licencias oficiales**, registros de derechos de autor ni materiales comerciales.
- El formato ChordPro se utiliza únicamente como **ayuda práctica** para la ejecución musical dentro del contexto de la misión.

Si algún autor, compositor o representante legal considera que algún contenido aquí alojado debe ser corregido, acreditado de forma distinta o retirado, puede contactarnos para atender la solicitud de buena fe.

## 2) ¿Cómo me preparo para usar este repositorio?

### 2.1) Resumen de configuración

- Los cantos están escritos en **ChordPro**.
- Los archivos pueden editarse directamente en GitHub o de forma local.
- La visualización puede hacerse mediante impresión o aplicaciones especializadas.

### 2.1.1) Entorno de edición recomendado

Estas opciones **no son obligatorias**, la estructura soporta otras herramientas de trabajo sin ningún problema.

- **Editor web de GitHub**: desde el repositorio, presiona la tecla `.` (punto) para abrir el editor estilo VS Code.
- **[Visual Studio Code](https://code.visualstudio.com/)** (recomendado)
  - Extensiones sugeridas:
    - **[ChordPro – VS Code Extension](https://marketplace.visualstudio.com/items?itemName=ricardomfmsousa.chordpro)**
      
    - **[YA ChordPro – VS Code Extension](https://marketplace.visualstudio.com/items?itemName=Homeputers.ya-chordpro)**

Estas extensiones permiten no solo editar el texto, sino también **previsualizar el resultado final** del canto.

### 2.1.2) Visualización de cantos

- **Impresión normal**, se requiere de una aplicación que lea los archivos.
- **Aplicación recomendada:** **[SongBook Pro](https://www.songbookproapp.com/)** (en su versión gratuita).

Características relevantes:
- Transposición de tonos
- Visualización clara
- Uso directo de archivos ChordPro

> Nota: SongBook Pro **consume** los archivos de este repositorio; **no es la fuente de verdad**.

## 4) Guía de contribución

Este repositorio **acepta contribuciones**, tanto de integrantes del ministerio como de colaboradores externos.

### 4.1) Reglas mínimas

- Este repositorio es la **fuente de verdad**.
- Evita modificar cantos existentes sin una justificación clara.
- Los cambios deben realizarse mediante **Pull Requests**.
- Usa nombres de archivos claros y consistentes.
- Respeta el formato ChordPro.

### 4.2) Coordinación

Si no estás seguro de un cambio, se recomienda **coordinar previamente** con el ministerio de música antes de modificar o agregar cantos.
